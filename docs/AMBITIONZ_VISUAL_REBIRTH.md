# Ambitionz Visual Rebirth

The new main product surface must feel cinematic, minimal, premium, dark, ritualistic and ownable.

## Visual Pillars

- Obsidian base.
- Ember and gold accents.
- Ritual-circle geometry.
- Premium Duel Altar layout.
- One huge Champion focus.
- Bound Soul orbs around the Champion.
- Ambition Core as a central vertical or circular meter.
- Clash axis between Champions.
- Compact Chronicle.
- Mobile-first hierarchy.

## Retired From Main Surface

- Three-lane board visuals.
- Generic monster rows.
- Old fantasy Arena class language.
- Dense dashboard cards.
- Generic neon sci-fi UI.
- Default prototype battler composition.

## Motion

Motion should clarify state changes: focus, strike, ascended, broken, committed. It must remain lightweight and respect reduced motion.
