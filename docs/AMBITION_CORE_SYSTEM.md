# Ambition Core System

The Ambition Core is the identity resource of Ambitionz.

## Sources

Ambition can be gained by:

- Choosing Focus.
- Burning cards into Echo.
- Receiving meaningful pressure.
- Champion, Bound Soul, Relic, Technique or Scheme effects.
- Winning specific Mind Clash patterns.

## Uses

Ambition can be spent to:

- Ascend the active Champion.
- Overrule future rule layers.
- Attempt Domination.
- Recover from pressure through card effects.
- Create rule-breaking turns as the architecture expands.

## Current Thresholds

- Ascension cards define their own Ambition cost.
- Domination currently requires 20 Ambition and an active Champion.
- Failed Domination spends the Ambition and applies vulnerability.

## Design Intent

Ambition is not generic mana or energy. It is psychological leverage. The player should feel it as a core filling with pressure, refusal and opportunity.
