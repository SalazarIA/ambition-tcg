# Ambitionz — Screenshot Guide

Use Android Studio emulator.

Recommended screenshots:
1. Landing/Home
2. Login/Register
3. Training/Arena
4. Deck Builder
5. Profile
6. Feedback

## Terminal Method
From project root:

adb exec-out screencap -p > mobile/AmbitionzAndroid/playstore/assets/screenshots/01-home.png

Repeat after navigating to each page.

## Play Console Note
Google Play accepts phone screenshots in PNG or JPEG.

Recommended:
- portrait screenshots
- actual app screens
- no browser address bar
- no fake UI
