# Ambitionz Android Debug Run

Version:
1.0.0-beta.1

Package:
com.ambitionzgame.app

Official URL:
https://ambitionzgame.com

Debug target:
Open the Android project in Android Studio and run on emulator or connected Android phone.

Expected behavior:
- App opens with Ambitionz name.
- Splash appears briefly.
- App loads https://ambitionzgame.com.
- Login, training, deck builder, shop, feedback and profile should work.
- Android back button should return within WebView navigation when possible.

Known requirement:
Android Studio or a valid Java Runtime is required for Gradle sync/build.
